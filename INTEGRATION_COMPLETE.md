
| Command | What It Does |
|---------|--------------|
| `python jarvis.py` | **Complete system** (all features) |
| `python jarvis.py --simple` | Simple demo (lightweight) |
| `python jarvis.py --enhanced-only` | JARVIS 2.0 only |

